# 31 숫자 자료형

## 311. 개요

1. 값이 숫자인데 ("", [], (), {} 등) 표시가 없다면 숫자 자료형으로 인식.
1. 소수점이 없으면 => `정수형` integer 의 약자 `int` 자료형. ( 1024 , 0 )
1. 소수점이 있으면 => `실수형` float point 의 약자 `float` 자료형. ( 3.141592 )

## 312. 숫자 자료형의 연산

| 산술 연산자 | 의미 |
|:----|:----|
| a + b | 더한다 |
| a - b | 뺀다 |
| a * b | 곱한다 |
| a / b | 나눈다 |
| a ** b | a 의 b 제곱을 구한다 |
| a // b | a 를 b 로 나누고 결과를 정수로 표시한다 |
| a % b | a 를 b 로 나눈 나머지를 구한다 |

| 비교 연산자 | 의미 | 비교 연산자 | 의미 |
|:----|:----|:----|:----|
| a > b | a가 b보다 크다 | a <= b | a가 b보다 작거나 같다 |
| a >= b | a가 b보다 크거나 같다 | a == b | a와 b가 같다 |
| a < b | a가 b보다 작다 | a != b | a와 b가 다르다 |

### 자료형 확인

```
num1 = 3
num2 = 3.5
print(type(num1))
print(type(num2))
```

# 32 문자열

## 321. 문자열 개요

따옴표로 둘러싸여 있으면 문자열.

## 322. 문자열 표현 방식

| 원하는 출력 문장 | 파이썬 코드 | 따옴표 사용 |
|:----|:----|:----|
| 강아지가 '멍멍' 짖는다 | print("강아지가 '멍멍' 짖는다") | 큰따옴표 |
| 강아지가 "멍멍" 짖는다 | print('강아지가 "멍멍" 짖는다') | 작은따옴표 |

### 오류인 경우

| 원하는 출력 문장 | 파이썬 코드 | 결과 |
|:----|:----|:----|
| 강아지가 "멍멍" 짖는다 | print("강아지가 "멍멍" 짖는다") | 오류 |
| 강아지가 "멍멍" 짖는다 | print('강아지가 "멍멍" 짖는다') | 정상 |

### 큰따옴표, 작은따옴표 3개 사용

| 파이썬 코드 | 원하는 출력 문장 |
|:----|:----|
| print("강아지가 '멍멍' 짖는다") | 강아지가 '멍멍' 짖는다 |
| print("그는 | 줄바꿈 오류 |
| 안녕이라고 | 여러줄의 문자열을 쓸때는 큰따옴표 3개를 써야 한다 |
| 인사한다") |  |

| 파이썬 코드 | 원하는 출력 문장 |
|:----|:----|
| print("강아지가 '멍멍' 짖는다") | 강아지가 '멍멍' 짖는다 |
| print("""그는 | 그는 |

| 파이썬 코드 | 원하는 출력 문장 |

| print("강아지가 '멍멍' 짖는다") | 강아지가 '멍멍' 짖는다 |
| print('''그는 | 그는 |

## 323. 이스케이프 코드

| 이스케이프 코드 | 설명 |
|:----|:----|
| \n | 줄바꾸기 |
| \t | 미리 설정된 고정 간격의 다음 탭으로 이동 |
| \\ | 문자열 안의 백슬래시 (\) 를 표현 |
| \" | 문자열 안의 큰따옴표 (") 를 표현 |
| \' | 문자열 안의 작은따옴표 (') 를 표현 |
| \b | 백스페이스 (커서 바로앞의 문자를 삭제) |
| \r | 커서를 문장 처음으로 이동 |

## 325. `in`, `not in` 연산자

- `in`: 문자열 안에 특정 글자 `있`으면 `True`, 없으면 `False'
- `not in`: 문자열 안에 특정 글자 `없`으면 `True`, 있으면 `False'

```
foo = "Hello World"
print("Hello" in foo)
print("abcd" in foo)
print("Hello" not in foo)
print("abcd" not in foo)
```

## 326. 문자열 관련 함수

### 문자열 글자수 세기: `len()`

```
foo = "Hello World"
result = len(foo)
print(result)
```

### 소문자를 대문자로 바꾸기: `upper()`

```
foo = "Hello World"
result = foo.upper() #-- upper(foo) 가 아님. 하다보면 기억이 된다
print(result)
```

### 대문자를 소문자로 바꾸기: `lower()`

```
foo = "Hello World"
result = foo.lower()
print(result)
```

### 문자열 안의 특정 글자 갯수 세기: `count()`

```
foo = "Hello World"
result = foo.count("o")
print(result)
```

### 문자열의 음절마다 구분자 끼워넣기: `join()`

```
foo = "대한민국"
result = "/".join(foo)
print(result)
```

### 문자열 양 끝의 공백 삭제하기: `strip()`

```
foo = "    Hello World     "
result = foo.strip()
print(result)
```

### 문자열 안의 단어 바꾸기: `replace()`

```
foo = "Hello World"
result = foo.replace("Hello", "Python")
print(result)
```

### 문자열 나누기: `split()`

```
foo = "Hello_World_and_Hello_Python"
result = foo.split("_")
print(result)
```

# 33 리스트와 튜플

## 331. 리스트

- 위인의 수 만큼 변수명을 만들어야 한다.

```
hero1 = "강감찬"
hero2 = "이순신"
hero3 = "안중근"
hero ...
```

- 리스트

`리스트 이름` = [ `원소1`, `원소2`, `원소3`, ... ]

```
hero = [ "강감찬", "이순신", "안중근", ... ]
```

원소의 구분은 인덱스 index 로 한다.

| hero | 강감찬 | 이순신 | 안중근 |
|:----:|:------:|:------:|:------:|
|인덱스| 0 | 1 | 2 |
|index| -3 | -2 | -1 |

`hero[0]` --> '강감찬' 임
`hero[-3]` --> '강감찬' 으로, [0]과 같은 위치임

## 332. 리스트 출력하기

### 원소 위치지정 꺼내기

- `[ ]` --> 인덱싱 연산자.
- 리스트를 선언할때 쓰는 `[ ]` 와는 경우가 다르다

```
guail3 = ["사과", "포도", "감", "바나나"] #-- 리스트 자료형으로 인식
print(guail3[0]) #-- guail3 변수의 몇번째 위치를 가리키는 것으로 인식
print(guail3[2])
print(guail3[-1])
print(guail3[-4])
```

| guail3 이 변수임을 확인 | print(`guail3`[0]) |
|:-----|:-----|
| guail3 변수의 내용으로 바꿔놓고 | print(`["사과", "포도", "감", "바나나"]` [0]) |
| 리스트 `[...]` 뒤에 `[0]` 인덱스 인식해서 | print(`"사과"`)  |

### 범위지정 원소 꺼내기 = 슬라이싱 slicing

- 리스트 이름 [ 시작 인덱스 : 끝 인덱스 ]
        - guail3[1:3] --> 1번원소 부터 3 바로 앞 원소까지 (즉, 1번, 2번 원소만 꺼낸다)
        - guail3[0:3] == guail3[:3] --> 0, 1, 2
        - guail3[2:4] == guail3[2:] --> 2, 3

## 333. 리스트의 연산

```
리스트1 + 리스트2 #-- 두 리스트 연결
리스트1 * 숫자 #-- 리스트1 을 숫자만큼 연결
값 in 리스트1 #-- 리스트1 의 원소 중에서 값이 `있`으면 True
값 not in 리스트1 #-- 리스트1 의 원소 중에서 값이 '없`으면 True
```

```
guail3 = ["사과", "포도", "앵두"]
guail4 = ["참외", "수박", "감"]
print(guail3 + guail4)
print(guail3 * 3)
print("포도" in guail3)
print("딸기" not in guail3)
print("포도@" in guail3)
print("포도@" in ["사과", "포도", "앵두"])
```

## 334. 리스트의 원소 추가, 삭제, 수정

> 리스트이름.`append`( 추가할 원소의 값 )
> 리스트이름.`remove`( 삭제할 원소의 값 )
> `del`( 리스트이름 [삭제할 인덱스번호] )

```
guail3 = ["사과", "바나나", "앵두", "참외""]
print(guail3)
guail.append["오렌지"] #-- 마지막에 추가
print(guail3)
guail.remove("바나나") #-- 원소의 값을 지정해서 삭제
print(guail3)
del(guail[2]) #-- 원소의 인덱스를 지정해서 삭제
print(guail3)
guail[0] = "딸기" #-- 원소를 직접 수정
print(guail3)
```

## 335. 리스트 관련 함수

### 리스트 원소 갯수 세기: `len()`

```
guail3 = ["사과", "바나나", "앵두", "참외""]
result = len(guail3)
print(result)
```

### 리스트 원소 정렬하기: `sorted()`

> `sorted(리스트이름)` #-- 오름차순 정렬
> `sorted(리스트이름, reverse = True)` #-- 내림차순 정렬

```
guail3 = ["사과", "바나나", "앵두", "참외""]
result = sorted(guail3, reverse = True)` #-- 내림차순 정렬
print(result)
```

### 원소의 값으로 인덱스 알아내기: `index()`

> `리스트이름.index(원솟값)`

```
guail3 = ["사과", "바나나", "앵두", "참외""]
result = guail3.index("앵두")
print(result)
```

## 336. 튜플 개요

> 튜플이름 = (원소, 원소, 원소, ...)

```
print("## 336. 튜플 개요 ----------")
foo1 = ("a", "b", "c")
foo2 = ("d", "e", "f")
print(foo1[0]) #-- 0번 원소
print(foo1[:2]) #-- 0, 1번 원소
print(foo1 + foo2) #-- 두 튜플 연결
print(foo1 * 2) #-- foo1 튜플 두번 연결
print(len(foo1)) #-- foo1 튜플의 원소 갯수
```

> 튜플은 원소를 추가, 삭제, 수정 안된다.

```
foo1 = ("a", "b", "c")
foo2 = ("d", "e", "f")
foo1.append("d") #-- 추가 안됨
foo1.remove("b") #-- 삭제 안됨
foo1[0] = "A" #-- 수정 안됨
```

# 34 딕셔너리

## 341. 딕셔너리 개요

> 딕셔너리 이름 = {키: 값, 키: 값, 키: 값, ...}

```
gagyok = {"짜장면": 6000, "짬뽕": 7000, "탕슉": 20000}
```

## 342. 딕셔너리 원소 추가,삭제,수정

> 딕셔너리이름[ 새로운 키 ] = 새로운 값

```
gagyok["짬뽕밥"] = 7500
```

> 리스트이름.remove( 삭제할 원솟값 )
> del( 리스트이름[ 삭제할 인덱스 ] )
> del( 딕셔너리이름[ 삭제할 키] )

```
gagyok = {"짜장면": 6000, "짬뽕": 7000, "탕슉": 20000}
print(gagyok["짜장면"])
gagyok["짬뽕밥"] = 7500
del(gagyok["짬뽕"])
gagyok["짜장면"] = 6500
print(gagyok)
```

## 343. 딕셔너리 관련 함수

### 딕셔너리 모든 키 가져오기: keys()

> 딕셔너리이름.keys()

```
guail7 = {"사과": 2500, "바나나": 3500, "수박": 3500}
result = guail7.keys()
print(result)
print(list(result)) #-- 딕셔너리를 리스트로 변환
```

### 딕셔너리의 모든값 가져오기: values()

> 딕셔너리이름.values()

```
guail7 = {"사과": 2500, "바나나": 3500, "수박": 3500}
result = guail7.values()
print(result)
```

### 딕셔너리의 모든 키,값을 쌍으로 가져오기: items()

> 딕셔너리이름.items()

```
guail7 = {"사과": 2500, "바나나": 3500, "수박": 3500}
result = guail7.items()
print(result)
```

## 352. 부울 자료형의 연산

| 사용 형식 | a | b | 결과 |
|:----|:----|:----|:----|
| a and b | True | True | True |
|  | True | False | False |
|  | False | True | False |
|  | False | False | False |
| a or b | True | True | True |
|  | True | False | True |
|  | False | True | True |
|  | False | False | False |
| not a or not b | True | - | False |
|  | False | - | True |
|  | - | True | False |
|  | - | False | True |

```
num1 = 20
num2 = 10
print(num1 >= 10 and num2 <= 30)
print(num1 == 20 or num2 == 5)
print(not num1 == 20)
```
