# 앱 비밀번호를 만드는 방법
## 앱 비밀번호를 생성하려면:
1.	프로필 및 설정 아바타에서 개인 설정 을 선택 합니다.
2.	액세스 관리  에서 앱 비밀번호 를 선택 합니다 .
3.	앱 비밀번호 만들기 를 선택 합니다 .
4.	앱 비밀번호에 비밀번호를 사용할 애플리케이션과 관련된 이름을 지정합니다.

이걸 가만히 두고볼 우리가 아니죠.

분노한 사람들이 저마다 의병이 되어, 온사방에 항일투쟁이 일어난 겁니다.
이때 왕산 허위선생이 군사정이 되어, 각지의 의병을 하나로 모으는 통합군을 만들었고,
일본군을 서울에서 밀어내기 위해 서울진격작전을 펼치게 됩니다.

그 의병군이 작전을 시작하기에 앞서, 한국에 있는 각국의 영사관에 선언문을 보내서
이번의 의병 전쟁이 광무황제의 칙령에 따른 한국의 독립전쟁임을 강조하고,
이는 당연히 국제법상 교전 단체이므로 전쟁에 관한 모든 법규가 적용되어야 한다고
주장했습니다.

네, 이로써 우리는 정식으로 선전포고를 한 것이고, 우리는 1945년 일본이 항복할 때까지
이 전쟁의 당사자였던 것입니다.

이때 어떤자가 묻기를, 조총을 가지고 일본군의 신식총을 대적하기 어려우니
기다려서 후일을 도모함이 옳지 않겠는가 하자,

선생께서는 무기의 장단을 비교함은 사안을 좁게 보는 것이니,
우리가 여기서 항쟁하여 필경은 목숨을 내어놓는 것이

적에게 떳떳하고 후손에게 큰 감화 교훈이 되리니

이나라를 장차 떳떳하게 세울 기초가 되는 것이라.
일을 넓게 보아서 이 자리를 피하려 하지 말라.
라고 하셨다고 합니다.
그누가 아무리 계란으로 바위를 치는 행동이라 비웃더라도
무력과 교묘한 사기질로 남을 겁박하는 자들의 삿된 행위들을
가차없이 부수어서 정의를 드러내려는 의로운 행동입니다.

저런 꿋꿋한 정신을 눈앞에서 보고 마음에 새긴 후손들이
이봉창이요, 윤봉길이요, 안중근이요, 또 다른 우리 조상님들이시죠.

우리가 민족주의를 주장한다고 생각한다면, 한참 멀리간 헛소리일 뿐입니다.
자기전에 잠깐만 하면서 왔다가 전에 본 글이 생각나 옮겨봅니다.

#!/bin/sh

ynow=$(date +'%y')
y2=$ynow

yy01="$((ynow - 1))01"
yy12="$((ynow + 1))12"

mnow=$(date +'%m')
m2="$((mnow + 99))" #-- 1...12월 => 100...111 --> 100 은 지난해의 지난달 112 로
if [ "$m2" = "100" ]; then
	m2="112"
	y2=$((y2 - 1))
fi
mm=${m2:(-2)} #-- 뒤에서 2자리 ; 지난달
ym=${y2}${mm}
if [ ! -f rsync_${ym}* ]; then
	#-- 지난달이 없으면,
	m2="$((mnow + 100))" #-- 1...12월 => 101...112
	mm=${m2:(-2)} #-- 뒤에서 2자리 = 이번달
	ym=${ynow}${mm}
fi

cat <<__EOF__

$(if [ -f rsync_${ym}* ]; then ls rsync_${ym}* ; fi)

----> Enter yymm: [ ${ym} ]
__EOF__
read a
if [ "x$a" = "x" ]; then
	a=${ym}
fi
echo "[ $a ]"

if [ $a -gt $yy01 ] && [ $a -lt $yy12 ]; then
	# if [ -f ksamlab_rsync_${a}.7z ]; then
	# 	echo "----> rm -f ksamlab_rsync_${a}.7z"
	# 	rm -f ksamlab_rsync_${a}.7z
	# fi
	#-- 두번이상 하면 7z 파일에 중복돼 만들어진다.
	echo "----> ls -l rsync_${a}* | 7za a -si ksamlab_rsync_${a}-$(date +'%y%m%d_%H%M%S').7z ; ls -l k* #-- 삭제할때, rm -f rsync_${a}*"
	ls -l rsync_${a}* | 7za a -si ksamlab_rsync_${a}-$(date +'%y%m%d_%H%M%S').7z ; ls -l k*
else
	echo "!!!! ----> ($a) 입력값이 년월의 범위 ${yy01} ... ${yy12} 를 넘어섭니다."
fi
