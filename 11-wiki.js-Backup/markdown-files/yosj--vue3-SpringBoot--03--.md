# 1: Vue 설치

1. 설치
```
npm install vue@next
```

2. 설치 확인
```
vue
```

# 2: 프로젝트 생성

1. 프로젝트 만들기
```
cd \Users\%username%
mkdir base-app
cd base-app
npm install
npm init @vitejs/app proj-vue01
```

2. 프로젝트 생성 화면
   프레임워크를 선택하는 화면이 나타나면, vue 프레임워크를 선택한다.
   자바스크립트 기반인 vue 를 선택한다.
   모듈 설치가 끝난 모습.

3. 프로젝트 실행
```
npm run dev
```
